Egyptian Treasures in Europe

STOP PRESS: the first museum CD, from Brussels, is now available and can be ordered directly from http://www.EgyptianTreasures.org

We are happy to announce that the first CD-ROM in the series "Egyptian Treasures in Europe" is now available for purchase directly from our web-site at

http://www.EgyptianTreasures.org

For your convenience the order page includes secure server facilities for secure credit card payment.

Hieroglyphic patterns

Charted Hieroglyph Pattern Set: This collection of detailed charted patterns can be used for many types of needlwork techniques. The set includes:

  • Patterns for four different color versions of each phonetic letter of the ancient Egyptian language, complete with name brand floss guide for color accuracy.
  • Image Diskette containing Bitmap images for each pattern for use in computer design
  • Separate patterns for writing letters from left to right and from right to left
  • Transliteration chart for design reference
  • Additional patterns: Cartouche (with reference information), scarab, and two border designs
